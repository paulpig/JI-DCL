# JI-DCL

## Reproducibility
Please follow steps below to reproduce the results of JI-DCL reported in our paper on the four datasets.
### LastFM
1. Run 
```python
  python main.py
```
2. Input
```python
  JI_DCL_lastfm
```
### Douban-Book
1. Run 
```python
  python main.py
```
2. Input
```python
  JI_DCL_Douban_book
```
### TrustFilm

  python main.py
